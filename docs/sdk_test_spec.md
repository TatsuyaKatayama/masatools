## 4. モック利用方針

- NATS Server および MinIO のモックを使用して、オフライン環境でのツールロジックを検証する。
- モックは以下の挙動を再現できること。
  - NATS: PENDING タスクあり／なし、切断、Rate Limit 超過
  - MinIO: 正常応答、アクセス拒否、帯域制限
